Vercel & Netlify deployment logs in your terminal

Filter by type (error/warning), export to JSON/CSV, real-time tail. All via direct API — no heavy UI needed.

Usage

$ npm i -g logv
$ export VERCEL_TOKEN=...
$ logv vercel --list
$ logv vercel -d dpl_xxx --filter error
$ logv netlify -d abc123 --tail

Why logv?

  • Fast — direct API calls, no heavy dashboard
  • Clear — filter by error/warning, readable output
  • Private — runs locally, no data sent to us
  • Cross-platform — macOS, Linux, Windows

Quick Install

npm i -g logv
export VERCEL_TOKEN=your_vercel_token
export NETLIFY_TOKEN=your_netlify_token
logv --help
